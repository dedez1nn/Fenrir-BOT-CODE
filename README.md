# Fenrir-BOT

Fenrir-BOT é um bot de Discord completo com sistema de economia próprio, progressão de níveis, guilds com raids, loja integrada e suporte a pagamentos via Pix pelo Mercado Pago. Projetado para proporcionar interações divertidas e competitivas dentro de servidores Discord.

## Funcionalidades

### Sistema de Economia e Progressão
- FenrirCoins: moeda própria do bot utilizada na loja.
- Experiência e níveis: cada usuário acumula XP ao interagir no servidor.
- Progressão de level: níveis desbloqueiam recompensas e vantagens.

### Guilds e Raids
- Criação e gerenciamento de guilds dentro do servidor.
- Sistema de raids entre guilds.
- Pontuação e XP próprios para cada guild.
- Rankings de guilds e usuários.

### Moderação e Automação
- Auto-remove de bots indesejados.
- Auto-delete de mensagens de convite para outros servidores.
- Comandos de administração e gerenciamento simplificados.

### Loja e Itens
- Loja interna utilizando FenrirCoins.
- Compra e venda de itens exclusivos.
- Itens com efeitos em XP, moedas ou interação em raids.

### Integração com Mercado Pago
- Pagamentos via Pix automático.
- Possibilidade de comprar FenrirCoins reais diretamente pelo bot.

### Outros Recursos
- Comandos de interação divertida para usuários.
- Sistema de logs e notificações de ações importantes.
- Estrutura modular para facilitar futuras adições de funcionalidades.
